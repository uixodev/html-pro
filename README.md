# HTML pro

> [!NOTE]
> **HTML**: HyperText Markup Language </br>
> **HyperText** - referes to links that connect wep pages to one another</br>
> **Markup** - highlighting the text

`< >` - these are called TAGS

### Semantic HTML

❌ **Non-semantic**: `<div>` and `<span>` → they don't tell what the content is, only that it's a generic container

✅ **Semantic**:`<header>`, `<nav>`, `<main>`, `<article>`, `<section>`, `<footer>`, `<form>`, `<table>`, `<button>` → give structure and meaning.
